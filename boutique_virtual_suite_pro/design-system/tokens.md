# Design Tokens

Primary UX targets: glass panels, smooth transitions, mobile bottom nav, desktop sidebar, light/dark.

## Colors

- Primary: `--bvp-primary` (moodboard-controlled)
- Accent: `--bvp-accent`
- Background: `--bvp-bg`
- Surface (glass): `--bvp-surface`

## Moods

- Festive: magenta/pink
- Party: violet/indigo
- Bridal: rose/red
- Office: slate/blue
- Traditional: amber/orange
