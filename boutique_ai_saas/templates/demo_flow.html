Step-by-step click checklist (Hindi)
  1. Home open → http://127.0.0.1:8000/ (Trending + vendors list show hona chahiye.)
  2. Login as customer → /accounts/login/ (ID: demo_customer)
  3. Vendor store open → /demo/
  4. Try-On upload → /demo/tryon/upload/ (Photo upload karke templates page par jao.)
  5. Live preview (no reload) → Templates page me template select karke preview dekho (JS overlay).
  6. Generate Try-On → “Generate Try-On” button (dummy compositor) → result page.
  7. Save look + Favorite → Favorite toggle / Save look buttons use karo, phir /favorites/ me verify karo.
  8. Order confirm → Try-On flow me “Confirm / Order” (vendor order create) karke order tracker/dash me check karo.
  9. Customer dashboard → /accounts/dashboard/ (Orders + wallet show hona chahiye.)
  10. Vendor dashboard → /vendor/ (Vendor modules + analytics widgets.)
  11. Tailor dashboard → logout karke tailor se login (ID: demo_tailor), then open /tailor/ (Tasks + payroll entries.)
  12. Magic Studio (React) → /magic/ (Try-On Studio + Magic Mirror + 3D mannequin + Saved looks + Admin in SPA)
Tip: Alag-alag roles test karne ke liye Incognito/Private window use karna easiest hai.
